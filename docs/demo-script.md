# Demo Script - Aurora Product Presentation

## Scenario 1: Live Fiverr Monitoring
- Open Fiverr in Chrome with extension installed
- Navigate to suspicious gig listing
- Point out Trust Gauge appearing in UI
- Show how threat level updates in real-time

## Scenario 2: Manual Content Analysis
- Open extension popup
- Paste suspicious seller message
- Click "Analyze"
- Show threat assessment with indicators
- Demonstrate Defense Narrative

## Scenario 3: Feedback Loop
- After analysis, show "Is this accurate?" feedback buttons
- Demonstrate how feedback improves future detections
- Explain ChromaDB learning

## Key Features to Highlight
1. **Real-time Detection**: Extension works instantly
2. **Transparent Analysis**: Users see reasoning behind alerts
3. **Adaptive Learning**: System improves from feedback
4. **User-Friendly**: Clear visual indicators
