Privacy-First Architecture = Sustainable Architecture

The same design decision that makes ShadowSense Aurora 100% private also makes it energy-efficient. Local inference eliminates idle server allocation, avoids data-center cooling overhead, and leverages hardware the user already owns — amortising embodied carbon at zero marginal cost.

🖥️
Zero-Cloud Inference

DeepSeek-R1 runs via Ollama on localhost:11434. No API key. No cloud endpoint. No data-center allocation.

💾
Local Vector Storage

ChromaDB persists scam embeddings to local disk with anonymized_telemetry=False. Zero external sync, zero transfer energy.

♻️
Hardware Reuse

Runs on an existing consumer GPU (RTX 3060/4060 class). Embodied carbon already paid — no new server rack required.
